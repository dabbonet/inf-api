API

API Address Format

Channel prefix: /{channel}/v1
Available channels: puter, codebuff, grok
NOTE
Claude Code users note: When configuring the API address, please use /{channel}， Do notadd /v1 suffix
Grok routing instructions: Grok is consistent with other channels, using /grok/v1 prefix.
Puter account instructions: Puter accounts currently only need to fill in auth_token。
Codebuff account instructions: Codebuff accounts currently use Bearer token; the upstream is invoked directly via paste-through.
INFO
Puter model instructions: Puter currently uses the "unprefixed main model" list from the reference repository, which already includes main model families such as Claude / OpenAI / Gemini / Grok / DeepSeek / Mistral; but like openrouter:、 togetherai: are temporarily not exposed directly to model management.
